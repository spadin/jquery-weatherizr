jQuery Weatherizr by spadin

Adds a CSS class to the body tag of your HTML describing the current weather so you may use CSS to style your homepage based on the weather.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/spadin/jquery-weatherizr

get the source code on GitHub : spadin/jquery-weatherizr
